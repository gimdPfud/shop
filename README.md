# 🛍️ Shop - 온라인 쇼핑몰 프로젝트
학원에서 배운 Spring Boot를 활용하여 제작한 온라인 쇼핑몰 프로젝트입니다.

### 소개
프로젝트명: Shop (온라인 쇼핑몰)
개발 형태: 수업 실습 프로젝트
개발 기간: -
목적: 웹 개발 전반적인 프로세스 학습
### 사용 기술
- Backend: Spring Boot
- Language: Java
- Database: MariaDB
- Frontend: HTML, CSS, JavaScript
- Template Engine: Thymeleaf
### 주요 기능
#### 상품 관리
- 상품 목록 조회
- 상품 상세 정보 보기
- 상품 등록/수정/삭제
#### 사용자 기능
- 회원가입 / 로그인
#### 쇼핑 기능
- 주문하기
- 주문 내역 확인하기
